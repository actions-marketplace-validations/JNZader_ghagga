GHAGGA combines static analysis, multi-model AI agents, and project memory to review your Pull Requests with context that improves over time. Free by default with GitHub Models — no API key needed.

Install GitHub App Documentation Open Dashboard

Static Analysis Trident

Semgrep, Trivy, and CPD scan for security issues, CVEs, and duplicated code — zero LLM tokens consumed.

3 Review Modes

Simple for speed, Workflow with 5 specialist agents, or Consensus with multi-model voting for critical code.

Project Memory

Learns patterns, decisions, and past bugs across reviews. PostgreSQL full-text search with privacy stripping.

Delegated Runner

Static analysis runs on your own GitHub Actions runner — 7GB RAM, unlimited free minutes for public repos. No server OOM.
